# Book of the Emerging Enterprise Landscape

This is the longform documentation of the Emerging Enterprise Landscape (EmELang), an abstract model of IT system management, intended as a common denominator to facilitate communication between teams of an enterprise and potentially even vendors of IT management software solutions.

## Usage

You need to install the Hugo CLI tool: see the [Hugo Installation instructions](https://gohugo.io/installation/)

* Start a local server for development: `hugo server`
* Build the complete website for distribution: `hugo build`
